Habilidades

Hard Skills

UI design é o processo de construção de interfaces com foco no usuário, aplicando conhecimentos sobre cores, tipografia, prototipagem, figma, etc.

HTML é uma linguagem baseada em marcação, onde marcamos os elementos para definir quais informações a página vai exibir.

CSS é uma linguagem de folha de estilo composta por “camadas”, criado com o propósito de estilizar as páginas HTML.

Sass é um pré-processador CSS. É uma simples linguagem de script usada em arquivos Sass.

O Git é um sistema de controle de versão que permite a criação e edição de arquivos. Já o GitHub é uma plataforma que armazena e/ou compartilha esses arquivos com a comunidade dev.

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

React é uma biblioteca JavaScript, desenvolvida pelo Facebook, para construção de interfaces de usuário de forma mais eficiente e componentizada.

O WordPress é um CMS que permite pegar qualquer site em HTML e torná-lo totalmente gerenciável, facilitando a vida dos clientes.

SEO é o conjunto de estratégias com o objetivo de potencializar e melhorar o posicionamento de um site nas páginas de resultados orgânicos.

Soft Skills

Adaptabilidade é a capacidade de lidar bem com situações novas ou cenários adversos, unindo a flexibilidade, coragem e a resiliência.

Atenção aos Detalhes é uma habilidade que permite concluir tarefas de trabalho com qualidade, precisão e consistência.

Colaboração é a capacidade de resolver coletivamente um problema ou fazer progresso em direção a um objetivo comum, sabendo lidar com as diferenças.

Criatividade é a habilidade de "pensar fora da caixinha", fazendo conexões entre assuntos aparentemente não relacionados e gerando soluções.

Comunicação Assertiva é a habilidade de saber se comunicar bem e ouvir atentamente o outro, sabendo contextualizar o que é transmitido.

Inteligência emocional é a capacidade de reconhecer e avaliar os sentimentos, tendo disposição e equilíbrio para lidar da melhor forma com eles.

Disposto a Aprender é ter a iniciativa em melhorar as habilidades atuais e aceitar novos desafios que possam acrescentar qualidade ao trabalho.

Gestão de Tempo é a habilidade de ajustar e organizar as diferentes atividades da rotina, sem comprometer diretamente a eficiência.

Ética é a capacidade de se comprometer consigo mesmo e com a empresa, sendo pontual, tendo compromisso com a verdade e pensando no coletivo.
